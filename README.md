# SB file share

## Features

- Chunked uploads (handles GB files)
- Upload progress
- Drag & drop
- Delete files
- Auto device discovery
- QR code join
- File expiration
- WebRTC client shared notes

## Next features

- WebRTC direct transfer between clients

  Single user → files stored on server
  Multiple users → direct WebRTC transfer
  Server used only for signaling

  This greatly reduces server bandwidth and enables faster LAN transfer.
